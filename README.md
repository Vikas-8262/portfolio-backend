# Vikas Rathod — Portfolio Backend API

Flask REST API for the portfolio site. Deployed on Render.com.

## Endpoints

| Method | Endpoint | Auth | Description |
|--------|----------|------|-------------|
| GET | `/` | No | Health check |
| POST | `/api/auth/login` | No | Admin login → returns JWT |
| POST | `/api/contact` | No | Send contact form email |
| GET | `/api/projects` | No | Get all added projects |
| POST | `/api/projects` | JWT | Add new project |
| DELETE | `/api/projects/:id` | JWT | Delete project |
| GET | `/api/experiences` | No | Get all added experiences |
| POST | `/api/experiences` | JWT | Add new experience |
| DELETE | `/api/experiences/:id` | JWT | Delete experience |

## Environment Variables (set on Render)

| Variable | Value |
|----------|-------|
| `GMAIL_USER` | vikas.rathod.tech@gmail.com |
| `GMAIL_APP_PASS` | your 16-char Gmail app password |
| `ADMIN_HASH` | SHA256 hash of your admin password |
| `JWT_SECRET` | auto-generated by Render |

## Local Development

```bash
pip install -r requirements.txt
python app.py
```

API runs at `http://localhost:5000`
